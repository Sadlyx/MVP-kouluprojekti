MS Office Tools

  • Word

    • Microsoft Word is a word processing program, which is available in several different language versions. It is user-friendly and easy to use. It also contains a wide range of features which require more skills to use.

  • Excel

    • Really high quality spreadsheet program, but at the same time extremely versatile and perhaps the most difficult and the most comprehensive to use of Office-programs.

  • Powerpoint

    • With Powerpoint program you can create high-quality tailor-made presentations. It offers design features that allow you to easily make visually better looking presentations. Its easy to use and and does not require lots of learning. For example interest calculation excel video is made using Powerpoint.

  • Outlook

    • It is e-mail programm combined with a clear calendar, contactlist and tool for managing files.

  • Visio

    • Visio is a diagramming and vector graphics application which is part of the Microsoft Office family, for example it is used to design network infrastructures.
